Welcome to my Automated SOC Environment

The goal with this project it to start from scratch and create a fully automated Security Operations Center Environment.

This project aims to handle SOC tier one analyst duties such as alert triage, log analysis, and data correlation. 

For this project, software such as Sysmon, Wazuh, and TheHive were integrated to handle the workload. Vultr utilized as the Cloud Service Provider. 

A workflow was created using Shuffle with the software integrated. 

The open source tool, Mimikatz is used to test the environment.
